欢乐连连看（LinePuppyLLK）
游戏指导书

----------------------------------------
一、软件说明
----------------------------------------
本程序为本地离线版，不依赖服务器。
账号、进度、排行榜和游戏设置均保存在当前 Windows 用户的本地 SQLite 数据库中。

----------------------------------------
二、发布包内容
----------------------------------------
发布包已包含程序运行所需的 Qt 运行时、插件和数据库驱动。
朋友的电脑上无需安装 Qt、VS Code 或数据库软件。

----------------------------------------
三、安装与启动
----------------------------------------
1. 下载并解压发布包。
2. 进入解压后的 LinePuppyLLK 文件夹。
3. 双击 appLinePuppyLLK.exe 启动程序。

如系统弹出安全提示，请选择“仍要运行”或“运行”。

----------------------------------------
四、数据保存
----------------------------------------
默认数据路径：
%APPDATA%\LLK\LinePuppyLLK\llk_local.sqlite

也可以在资源管理器地址栏输入：
%APPDATA%\LLK\LinePuppyLLK

进入后即可看到本地数据库文件 llk_local.sqlite。

----------------------------------------
五、备份与重置
----------------------------------------
备份方法：复制 llk_local.sqlite 即可。

重置方法：关闭程序后删除 llk_local.sqlite，再重新启动程序。

----------------------------------------
六、使用说明
----------------------------------------
1. 首次登录或注册后，请保留用户名和密码。
2. 游戏数据不会跨电脑同步，每台电脑保存各自的本地数据。
3. 如果删除程序文件，通常不会影响用户目录中的本地数据文件。

----------------------------------------
七、故障处理
----------------------------------------
1. 程序无法启动
- 请确认运行的是 appLinePuppyLLK.exe。
- 请确认发布包内的 platforms、sqldrivers 和 translations 目录完整。

2. 更换电脑后数据不见了
- 这是正常现象，因为本版本为本地离线版，不会自动同步数据。

3. 程序闪退或打不开
- 请先确认发布包文件未缺失。
- 请把截图发给我，以便进一步确认问题。

----------------------------------------
八、内测提示
----------------------------------------
如果想重置进度，可以把本机的 llk_local.sqlite 删除后重新启动程序。
